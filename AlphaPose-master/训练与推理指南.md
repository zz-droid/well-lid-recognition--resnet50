# AlphaPose 训练与 Pose 连线 完整指南

使用 data 中的 CrowdPose 数据集，ResNet50 网络训练，并实现 pose 骨架连线可视化。

---

## 第一步：准备数据集

### 1.1 下载 CrowdPose 图像

当前 `data/crowdpose/annotations/` 下已有标注文件，但需要下载图像。

1. 访问 [CrowdPose 官网](https://github.com/Jeff-sjtu/CrowdPose) 或 [百度网盘](https://pan.baidu.com/s/1COQ5s0s6b0U2bJ0X0X0X0) 下载图像
2. 解压后目录结构应为：
   ```
   data/crowdpose/
   ├── images/
   │   ├── train/    # 训练图像 (103492.jpg 等)
   │   └── val/      # 验证图像
   └── annotations/
       ├── crowdpose_train.json
       └── crowdpose_val.json
   ```

### 1.2 检查标注路径

标注中 `file_name` 格式为 `103492.jpg`，图像路径应为：`ROOT + IMG_PREFIX + file_name`

---

## 第二步：修改配置文件

编辑 `configs/coco/resnet/256x192_res50_lr1e-3_2x-dcn1.yaml`，确保路径正确：

```yaml
DATASET:
  TRAIN:
    TYPE: CrowdPose
    ROOT: D:/AlphaPose-master/AlphaPose-master/data/crowdpose
    IMG_PREFIX: images/train
    ANN: annotations/crowdpose_train.json
  VAL:
    TYPE: CrowdPose
    ROOT: D:/AlphaPose-master/AlphaPose-master/data/crowdpose
    IMG_PREFIX: images/val
    ANN: annotations/crowdpose_val.json
```

**注意：** `ROOT`、`IMG_PREFIX`、`ANN` 使用相对路径，最终路径 = `ROOT/IMG_PREFIX/xxx.jpg` 和 `ROOT/ANN`

---

## 第三步：安装依赖

```powershell
cd D:\AlphaPose-master\AlphaPose-master
pip install -e .
pip install tensorboardX pycocotools
```

**Windows 注意：** 若 `pycocotools` 安装失败，使用：
```powershell
pip install git+https://github.com/philferriere/cocoapi.git#subdirectory=PythonAPI
```

---

## 第四步：训练模型

```powershell
cd D:\AlphaPose-master\AlphaPose-master
python train.py --cfg configs/coco/resnet/256x192_res50_lr1e-3_2x-dcn1.yaml --exp-id crowdpose_res50
```

- **有 GPU：** 自动使用 CUDA 训练
- **无 GPU：** 需修改 `train.py` 中的 `.cuda()` 为 `.to(opt.device)`（CPU 训练很慢）

训练完成后，模型保存在：`./exp/crowdpose_res50-256x192_res50_lr1e-3_2x-dcn1.yaml/model_*.pth`

---

## 第五步：推理并实现 Pose 连线

### 5.1 单张图片推理（带骨架连线）

```powershell
python scripts/demo_inference.py --cfg configs/coco/resnet/256x192_res50_lr1e-3_2x-dcn1.yaml --checkpoint ./exp/crowdpose_res50-256x192_res50_lr1e-3_2x-dcn1.yaml/final.pth --image examples/demo.jpg --outdir examples/res/ --save_img --vis
```

- `--save_img`：保存带骨架连线的结果图
- `--vis`：显示可视化窗口

### 5.2 文件夹批量推理

```powershell
python scripts/demo_inference.py --cfg configs/coco/resnet/256x192_res50_lr1e-3_2x-dcn1.yaml --checkpoint ./exp/crowdpose_res50-256x192_res50_lr1e-3_2x-dcn1.yaml/final.pth --indir examples/demo/ --outdir examples/res/ --save_img
```

### 5.3 视频推理

```powershell
python scripts/demo_inference.py --cfg configs/coco/resnet/256x192_res50_lr1e-3_2x-dcn1.yaml --checkpoint ./exp/crowdpose_res50-256x192_res50_lr1e-3_2x-dcn1.yaml/final.pth --video your_video.mp4 --outdir examples/res/ --save_video --save_img
```

## 第六步：Pose 连线说明

AlphaPose 的 `alphapose/utils/vis.py` 已实现骨架连线逻辑：
- 根据关键点数量自动选择连线方式（17/14/136 等）
- CrowdPose 14 关键点：head-neck-shoulders-arms-hips-legs 等骨架连线
- 输出图像中会绘制关节点和骨架线

若 14 关键点连线异常，需在 `vis.py` 中增加 `kp_num == 14` 的 `l_pair` 定义（CrowdPose 骨架连接）。

---

## 常见问题

| 问题 | 解决 |
|------|------|
| GPU 未检测到 | 安装 CUDA 版 PyTorch；或修改代码支持 CPU |
| 数据集路径错误 | 检查 ROOT/IMG_PREFIX/ANN 组合后的路径 |
| 训练报错 KeyError | 检查 MODEL.TYPE、DATASET.TYPE 与类名一致 |
| 推理无连线 | 使用 `--save_img` 或 `--vis` 参数 |
